為什麼選擇 AWS？

Amazon Web Services (AWS) 是全球最全面且獲廣泛採用的雲端。數百萬客戶，包括成長最快的新創公司、最大型企業以及領先的政府機構都使用 AWS 來變得更靈活、降低成本且更迅速地創新。

最廣泛且最深入的雲端功能集

AWS 為您提供最好的選擇和靈活性，以滿足您的特定需求，因此您可以為工作選擇合適的工具。AWS 提供最廣泛的運算執行個體、儲存類別、資料庫和分析，全部專為提供最佳成本和效能而設計。這包括最全面的 AI 和資料服務集，包括您可選擇用於建置生成式 AI 應用程式的領先 FM。藉助 AWS，您可更快速、更輕鬆且更具成本效益地將現有的應用程式遷移至雲端，或者盡情打造心目中的理想作業環境。

最大的客戶和合作夥伴社群

AWS 擁有最大規模的社群，並且在全球 200 多個國家/地區內擁有數百萬作用中客戶和 130,000 多名 AWS 合作夥伴。客戶遍佈幾乎每個產業，規模不等，包括新創公司、企業和公共部門組織，他們在 AWS 上執行每個可想象的使用案例。AWS 合作夥伴網路包括數千位擅長 AWS 服務的系統整合商和成千上萬採用其技術在 AWS 上運作的獨立軟體開發廠商 (ISV)。

進一步了解我們的合作夥伴

您可以信任的安全性

安全性是我們的首要考量。AWS 的架構設計是當今最靈活且最安全的雲端運算環境，其基礎設施旨在滿足最敏感組織 (包括政府、醫療保健和金融服務) 的安全需求。該基礎設施由一組深入的雲端安全工具提供支援，具有超過 300 項安全、合規性，以及控管各項服務與功能，並支援 143 項安全性標準和合規性認證。在以安全為導向設計的雲端上建置、執行和擴展應用程式。

進一步了解我們的安全性

加速轉型的創新

AWS 從客戶的需求出發開展工作，不斷創新以協助您將想法變為現實，並且加速目前和未來的轉型。我們是雲端運算的先驅，並投入了多年的時間，設計針對雲端最佳化的自訂晶片，從而為各種應用程式和工作負載提供最佳的價格效能比和能源效率。AWS 培養創新文化，以促進客戶成長、改善效能並降低成本，讓您可以專注於創新。 

進一步了解 AWS 可如何協助您創新

經過驗證的操作技能

AWS 擁有您企業可依賴的無與倫比的經驗、可靠性、安全性和效能。AWS 擁有最深入的大規模營運經驗，與位居下一位的雲端供應商相比，資料中心數量超過 3 倍。自 2006 年以來，AWS 已為全球數百萬客戶提供雲端服務，這些客戶信任我們可靠且可擴展的基礎設施，為其最重要的應用程式和最嚴苛的工作負載提供服務。
